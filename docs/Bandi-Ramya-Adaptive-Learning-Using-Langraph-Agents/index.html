🧠 Adaptive Learning with LangGraph Agents

Developer: Ramya Bandi

📌 Project Overview

This project explores modular agent design using LangGraph, integrating student input, code execution, and mastery tracking with fallback handling and UI refinement.

🚀 Key Features

  • LangGraph agent orchestration
  • Fallback error resilience
  • Interactive UI with Panel
  • Tracer logic for mastery evaluation

🛠️ Technologies

  • Python, Panel
  • LangGraph, Pydantic
  • GitHub Actions, VS Code Dev Containers
